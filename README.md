# 📖 Caminho Diário 

> Uma aplicação interativa de devocional e estudo bíblico que utiliza lógica de "IA Local" para gerar desafios de fé, incentivando o hábito da leitura através de gamificação.

[![GitHub license](https://img.shields.io/github/license/devgui324/caminhodiario)](https://github.com/devgui324/caminhodiario)
[![PRs Welcome](https://img.shields.io/badge/PRs-welcome-brightgreen.svg)](https://github.com/devgui324/caminhodiario/pulls)

---

## ✨ Funcionalidades

* **🧠 Desafios Gerados por IA:** O sistema combina ações, alvos e temas para criar missões únicas a cada clique ou a cada dia.
* **🏆 Sistema de Gamificação:** Ganhe pontos ao concluir desafios (+10) ou salvar aprendizados (+5). Evolua sua patente de *Servo Esforçado* até *Ancião da Fé*.
* **📚 Consulta Bíblica Multi-versões:** Suporte para diferentes traduções (NVI, ARA, ACF) carregadas via JSON.
* **✍️ Registro de Aprendizado:** Salve suas reflexões vinculadas diretamente ao versículo estudado com opção de excluir registros.
* **🙏 Mural de Oração:** Espaço dedicado para registrar e acompanhar seus pedidos de oração.
* **🌗 Tema Dinâmico:** Suporte a Modo Claro e Escuro para uma leitura confortável em qualquer ambiente.
* **💾 Gerenciamento de Dados:** Sistema de Backup (Exportar/Importar JSON) com alerta de lembrete e função de Reset total.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5:** Estrutura semântica organizada por abas e cards.
* **CSS3:** Estilização moderna com Variáveis CSS, Flexbox e animações de Splash Screen.
* **JavaScript (ES6+):** Motor de geração aleatória, manipulação de DOM e lógica de estados.
* **LocalStorage:** Armazenamento local persistente para pontos, histórico e preferências.

## 🛡️ Backup e Privacidade
Seus dados são 100% privados. Nenhuma informação de oração ou aprendizado é enviada para servidores externos. Tudo fica salvo no seu próprio dispositivo.

O sistema emitirá um alerta visual caso você fique mais de 7 dias sem realizar um backup manual.
